## рҹ“Ң Overview

**KrishiSahay** is an AI-powered agricultural assistant designed to help farmers make informed farming decisions through intelligent conversations. The chatbot provides instant answers to agriculture-related questions, supports multiple languages, offers weather-based farming insights, and assists in identifying crop diseases through image analysis.

The goal of KrishiSahay is to bridge the gap between modern technology and agriculture by making expert farming guidance accessible, simple, and available anytime.
